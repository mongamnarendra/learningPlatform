AI/ML Interview FAQs

Supervised Learning

1. What is the difference between supervised and unsupervised learning?
2. How do you handle overfitting in machine learning models?
3. What is the bias-variance tradeoff?
4. Explain the workings of a decision tree algorithm.
5. How does regularization (like L1 and L2) help in machine learning models?
6. What is the difference between classification and regression in supervised learning?

Unsupervised Learning

1. What is the K-Means clustering algorithm, and how does it work?
2. How do you determine the optimal number of clusters in K-Means?
3. What is the difference between K-Means and hierarchical clustering?
4. Explain Principal Component Analysis (PCA) and its applications.
5. How would you handle dimensionality reduction in large datasets?
6. What are some challenges of unsupervised learning compared to supervised learning?

Reinforcement Learning

1. What is reinforcement learning, and how is it different from supervised learning?
2. Explain the exploration vs. exploitation dilemma in reinforcement learning.
3. What is Q-learning, and how does it work?
4. Can you explain the reward function in reinforcement learning?
5. What is a Markov Decision Process (MDP)?
6. How do policy-based methods differ from value-based methods in reinforcement learning?

Natural Language Processing (NLP)

1. What is the difference between stemming and lemmatization?
2. Explain how word embeddings like Word2Vec or GloVe work.
3. What is the purpose of TF-IDF in text analysis?
4. How do transformers improve NLP tasks like translation and summarization?
5. What are some common applications of NLP?
6. How does Named Entity Recognition (NER) work in NLP?

Deep Learning

1. What is a neural network, and how does it function?
2. Explain the vanishing gradient problem in deep learning.
3. What is a convolutional neural network (CNN), and where is it used?
4. What is the purpose of an activation function in a neural network?
5. Explain the difference between CNNs and RNNs.
6. What is dropout in deep learning, and why is it used?

Model Evaluation and Validation

1. What is cross-validation, and why is it used?
2. Explain precision, recall, and the F1-score in classification models.
3. What is the ROC curve, and what does AUC signify?
4. How do you handle imbalanced datasets in classification problems?
5. What is overfitting, and how can it be prevented?
6. What is a confusion matrix, and how is it useful in evaluating classification models?

Optimization Algorithms

1. What is gradient descent, and how does it work?
2. Explain the difference between batch gradient descent and stochastic gradient descent.
3. How does the Adam optimizer improve over standard gradient descent?
4. What is the role of the learning rate in optimization?
5. How would you prevent a model from getting stuck in local minima?
6. What is momentum in gradient descent, and how does it help optimization?
